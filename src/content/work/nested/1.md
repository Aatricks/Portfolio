Julia Set Explorer grew beyond a single script. The repository now carries several implementations of the same fractal workflow, each useful for a different style of use.

- the Python CLI uses Numba-parallel computation, cached renders, and scripted GIF generation for zoom and parameter-sweep outputs
- the PyQt interface adds interactive exploration with sliders, zoom controls, formula variants, colormap changes, and export tools
- the Rust version provides a parallel command-line renderer with `clap`, `rayon`, and progress-bar feedback
- there is also a lightweight C implementation and Docker helpers, which makes the repo part experiment playground and part performance comparison

The repo ends up feeling more like a small fractal lab than a single implementation, which is exactly why it is still worth showing.
