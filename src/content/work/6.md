`CChess` is a compact command-line engine where almost everything lives in plain C rather than behind helper libraries or frameworks.

- `board.c` handles move legality, attack detection, castling, promotion, evaluation tables, and the minimax search loop
- the evaluation path uses both material values and piece-square tables instead of only counting pieces
- the search uses alpha-beta pruning with a fixed depth, which keeps the code understandable while still producing a usable opponent
- `main.c` stays intentionally simple: print the board, let the engine move, accept the human move, repeat

For me, it works as a compact example of direct, inspectable algorithmic code.
