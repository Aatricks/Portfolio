Polytech Sous-Marin is a hardware-software build rather than a software-only demo. The repository combines the printed parts and the Arduino control sketch in one place.

- the repo includes STL files for the body and mechanical parts, not just control code
- `main.ino` drives four DC motors, listens to an HM10 BLE module, and maps incoming commands to propulsion changes
- the MPU6050 path includes sensor calibration, periodic reads, and a Kalman filter for angle estimation
- the sketch coordinates motor timing and telemetry in the main loop instead of treating sensing and control as unrelated pieces

The value here is the integration work: electronics, control, and fabrication all had to line up for the prototype to move at all.
