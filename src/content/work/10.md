EasyReader is an offline-first Android reader for novels, manga, manhwa, and local documents built like a full Android product with local intelligence.

### Architecture & Constraints
Compose navigation, Room persistence, Hilt DI, source adapters, local-file ingestion, and on-device summarization all feed the same reading flow.
- `MainActivity` routes between reader, library, and explore screens while handling deep links, shared URLs, and local document picks.
- `ContentRepository` normalizes four content paths: remote web chapters, PDFs, EPUBs, and local HTML files, with cache inspection and prefetch support.
- `SummaryService` integrates `llmedge` to download a local Qwen3 GGUF model and generate chapter summaries on-device instead of sending text to a remote service.

This project demonstrates the ability to bridge local models with persistent state and user-centric design, proving I can build software that users actually live in, not just backend tooling.
